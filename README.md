# Contact Management App with Chart and Map

## Tech Stack Used

- React
- React-Dom
- Redux
- Redux toolkit
- React Query
- React Leaflet
- Iconify
- Tailwind CSS
- TypeScript
- Chart Js

## Steps to run this app

- Clone this repository.
- Run the command `npm install`
- Then `npm run start`
